I want to build a professional full-stack web application called "PortfolioPilot" that serves as an AI-powered investment research and portfolio recommendation platform.

The goal is NOT to predict stock prices.

The goal is to help investors build better long-term portfolios based on their investing style.

The website should look modern, premium, and similar to Bloomberg Terminal meets Apple.

Use:
- React
- TypeScript
- Tailwind CSS
- Node.js backend
- Express
- PostgreSQL (or Supabase)
- Chart.js or Recharts
- Alpha Vantage, Finnhub, Polygon.io, or Yahoo Finance APIs
- OpenAI API only for natural-language explanations (never for stock selection)
